Five-step learner workflow

Keep drafting tools flexible while centralizing evidence and feedback. Supabase stores the official record so students, teachers, and LLM co-pilots stay in sync.

Check the brief

Students sign in with Supabase magic links and review the weekly assignment plus prompt starters for their preferred LLM.

Draft anywhere

Learners compose offline, in Docs, or on paper while logging time spent and saving transcripts from any external model.

Upload evidence

Final draft, transcript, and reflection land in Supabase storage with automatic word counts and streak updates.

Review together

Teacher feedback, rubric scores, and audio notes appear instantly in the student timeline inside the GitHub Pages dashboard.

Share with an LLM

Students generate a signed digest link to paste into their LLM so each coaching session starts with accurate context.

Shared progress views

Everyone sees the same evidence trail—students, teachers, and guardians if invited.

  1. Student timeline

    Visualize submissions, reflections, and streaks with toggles for drafts, transcripts, and teacher responses.

  2. Teacher console

    Filterable class board highlights late work, target CEFR levels, and open comments across all students.

  3. Progress digest

    Generate a Supabase-signed JSON summary so the next LLM session adapts to current strengths and goals.

LLM-ready digest in three safeguards

Give language models just enough context to coach effectively without sharing entire drafts by default.

  1. Edge function filter: progress_digest returns goals, recurring issues, and next steps—not raw documents.
  2. Expiring signature: Copy links expire within an hour and can be revoked instantly from the dashboard.
  3. Student control: Learners decide which metrics appear in the digest before sharing with an LLM.

Stack essentials

Everything runs on free tiers—deploy once and invite the class to sign in.

  • GitHub Pages hosts the static dashboard.
  • Supabase handles auth, database, and storage.
  • Row-level security keeps data scoped per user.
  • GitHub Actions cron pings Supabase for weekly summaries.
